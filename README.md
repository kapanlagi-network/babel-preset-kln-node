Opinionated Babel preset
=======================================


## Installation

Using [npm](https://www.npmjs.com):

```
npm install babel-kln-preset-node
```


## Documentation

Add `babel-kln-preset-node` to your [Babel](http://babeljs.io/) configuration :

```js
{
  "presets": [
    "kln-node"
  ]
}
```
